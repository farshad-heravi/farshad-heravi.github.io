SMA-actuated Morphing-Wing Mechanism

In this mechanism, a Shape Memory Alloy (SMA) wire was used to actuate the mechanism. An Adaptive Neuro-Fuzzy Inference System (ANFIS) was used to model the hysteresis behavior of the SMA wire. The obtained model as well as designed controller were integrated in a LabView model, as shown in the following picture. An adaptive controller was used to perform the position control of the mechanism.
