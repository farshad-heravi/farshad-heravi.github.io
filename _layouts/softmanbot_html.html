Integration of a Cartesian Robot in a Robotic Cell

Within this project, I have worked on the development of a ROS-based package to integrate an igus cartesian robot into a robotic cell consists of two UR5e and a cartesian robots along with a sewing machine. The project, SoftManBot, aimed to automate a sewing process of a product. I have used C++ to develop a HardwareInterface for the 3-axis cartesian robot for the ROS. The interface should handled two motors of the cartesian robot as well as a Maxon motor as a custom third link. The developed package is available in github page.
