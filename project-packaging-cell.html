Autonomous Dual-Arm Packaging Cell

A full-stack ROS2 and Isaac Sim simulation for industrial automation.

Timeline: 2025
Type: Simulation
Role: Sole Developer (A-Z)

Project Overview

The Goal

The aim was to develop a dual-arm robotic cell which can autonomously work at the final stage of a production line to package bottle sprayers in dedicated containers.

My Role

I implemented the project from A to Z, including designing the robotic cell, creating the simulation environment in IsaacSim, integrating it with ROS2, and implementing motion planning for both arms using MoveIt2.

Key Outcomes

  • Successfully simulated a complete, end-to-end packaging workflow.
  • Developed a collision-free motion plan for a complex dual-arm setup.
  • Created a robust and scalable task management system using Behaviour Trees.

Tech Stack

ROS2 Humble Isaac Sim MoveIt2 C++ Behaviour Trees Python

Technical Deep Dive

Dual-Arm Motion Planning

Coordinating two robotic arms in a shared workspace without collision is a significant challenge. I utilized the MoveIt2 framework to handle the complex motion planning. For generating paths in the cluttered environment, I leveraged sampling-based planners like RRTConnect, which excels at finding valid paths in high-dimensional spaces. For the final, precise movements such as approaching the box or placing the sprayer, the Pilz Industrial Motion Planner was used to generate smooth, predictable Cartesian trajectories (LIN and PTP), ensuring the reliability required in industrial tasks.

Task Management with Behaviour Trees

To manage the complex sequence of actions (folding the box, picking, placing, sealing), I implemented a Behaviour Tree (BT). BTs are ideal for robotics as they provide a modular and reactive way to define task logic. The tree was structured with high-level sequences like "AssembleBox" and "PackItems", which were composed of smaller, reusable actions. This approach made the system robust and easy to debug, as the state of the robot's logic was always clear and could react to sensor feedback or failures in a predictable manner.
